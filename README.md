# 🧮 Simple Calculator (GUI)

A basic desktop calculator application built using **Python Tkinter** that performs arithmetic operations with a user-friendly interface.

## 🚀 Features
- Perform addition, subtraction, multiplication, and division
- Input validation and error handling (e.g., divide by zero)
- Clean and simple GUI using Tkinter

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)

## ▶️ How to Run
1. Install Python
2. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/calculator-app.git

3. Run the program:
   python calculator.py
